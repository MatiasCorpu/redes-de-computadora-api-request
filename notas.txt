    post /ticket

INVENTORY:<<<<<<<<<<<<<<<

ADD/DELETE
addNetworkDevice-> Adds the device with given credential
    post /network-device

deleteNetworkDeviceById->Removes the network device for the given ID
    delete /network-device/{id}

deleteNetworkDeviceByIp->Removes the network device for the given IP address
    delete /network-device/ip-address/{ipAddress}


GET
getNetworkDeviceById->Gets the network device for the given device ID
/network-device/{id}

getNetworkDeviceByIp->Gets the network device with the given IP address
/network-device/ip-address/{ipAddress}

getNetworkDeviceCount->Gets the count of network devices filtered by management IP address, mac address, hostname and location name.
/network-device/count

getNetworkDeviceRange->Gets the list of network devices for the given range.
/network-device/{startIndex}/{recordsToReturn}

getNetworkDevices->Gets the list of network devices. Network-Device List includes unsupported, managed, and unreachable. Devices with an unknown type due to unreachable will appear in the network-device list
/network-device

updateNetworkDevice->Sync the devices provided as input
/network-device
